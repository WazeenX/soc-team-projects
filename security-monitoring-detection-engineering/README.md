# Security Monitoring & Detection Engineering

This project demonstrates the implementation of core Security Operations Center (SOC) monitoring and detection workflows within a controlled lab environment. Using SIEM technologies, network monitoring tools, and endpoint visibility solutions, the project focuses on collecting, analyzing, and correlating security events to improve threat detection capabilities.

The objective of this project is to develop practical experience with log management, security monitoring, alert engineering, and detection analysis through hands-on activities and documented assessments.

## Overview

Security monitoring and detection engineering are fundamental components of modern SOC operations. In this project, multiple security tools were used to collect, centralize, and analyze security telemetry from various sources.

Activities included log ingestion, dashboard creation, alert rule development, network monitoring, vulnerability assessment, endpoint visibility, and security event documentation. The project demonstrates how security teams can transform raw security data into actionable alerts and operational intelligence.

---

## Tools & Environment

**Elastic Security** — Used for log management, event correlation, and dashboard visualization.

**Wazuh** — Used for security monitoring, alert generation, and detection engineering.

**Snort** — Used for network intrusion detection and traffic monitoring.

**Nessus** — Used for vulnerability assessment and security scanning.

**Osquery** — Used for endpoint visibility and system investigation.

**Windows Event Viewer** — Used for log analysis and event investigation.

**Virtual Lab Environment** — Provides an isolated environment for safe security testing and monitoring.

## Environment

All activities were performed within isolated virtual lab environments for educational and defensive security purposes.
